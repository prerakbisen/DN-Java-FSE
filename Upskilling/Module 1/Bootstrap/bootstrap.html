15. Sass Customization Notes

Sass Setup

To customize Bootstrap using Sass, install Bootstrap using npm:

npm install bootstrap
npm install sass

Create a Sass file and override variables before importing Bootstrap:

$primary: #6f42c1;
$border-radius: 1rem;
@import "node_modules/bootstrap/scss/bootstrap";
